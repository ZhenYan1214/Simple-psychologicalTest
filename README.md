## Enjoying~
這是一個AI心理測驗生成器，讓你可以透過一系列問題來探索你的內心世界！利用簡單的HTML和JavaScript達到互動式測驗，快速了解自己的心理特質~  

# 使用方法
把這個專案下載到你的電腦  
打開 index.html 檔案，開始測驗  
回答問題，看看AI會給你什麼有趣的結果！  
快來試試看，發現你心裡的小秘密吧！✨
