# 🧠 AI 心理測驗生成器

---

## 📖 介紹  
這是一個簡單有趣的 **AI 心理測驗生成器**，讓你可以透過回答一系列問題，探索自己的內心世界！專案利用 **HTML** 和 **JavaScript** 實現互動式測驗，讓你快速了解自己的心理特質。  
✨ **快來試試看，挖掘你的內心秘密吧！**  

---

## 🎮 開始測驗

1. 打開 `index.html` 檔案，開始測驗。  
2. 按照測驗提示回答問題，查看 AI 給出的有趣結果！
